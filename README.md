# Global Food Security Dashboard

An interactive Streamlit dashboard analysing the FAO Suite of Food 
Security Indicators for 198 countries from 2000 to 2024.

Built for 5DATA004C Data Science Project Lifecycle- University of Westminster.

## Dataset
FAO Suite of Food Security Indicators — World Bank Data360
https://data360.worldbank.org/en/dataset/FAO_FS

## Live App
[View the dashboard here](YOUR STREAMLIT LINK)

## Dashboard Features
- World choropleth map of food security indicators
- Trend analysis with COVID-19 impact chart
- Country rankings and progress comparison
- Nutrition profile with 4 dimensions of food security
- GDP vs hunger scatter plot
